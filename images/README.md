# 📸 Imagini Galerie

Adaugă poze în folderele corespunzătoare:

- **pregătiri/** - Machiaj, ținută, pregătiri
- **ceremonie/** - Momentele ceremoniei
- **petrecere/** - Cina și petrecere
- **detalii/** - Decorații, flori, detalii

Apoi actualizează `data.js` cu lista imaginilor.
